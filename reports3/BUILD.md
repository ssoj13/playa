# Build Instructions

## ВАЖНО: Сборка проекта

**⚠️ ВСЕГДА используйте `start.cmd` для сборки проекта!**

```cmd
start.cmd
```

### Почему?

- Проект требует специальной настройки окружения
- Прямой вызов `cargo build` не сработает
- `start.cmd` настраивает все необходимые переменные и зависимости

### Команды

```cmd
# Сборка
start.cmd

# Запуск
cargo run

# Тесты
cargo test
```

### НЕ ДЕЛАЙТЕ ТАК:

```cmd
# ❌ НЕ РАБОТАЕТ
cargo build

# ❌ НЕ РАБОТАЕТ
pwsh.exe -Command "cargo build"
```

### ДЕЛАЙТЕ ТАК:

```cmd
# ✅ ПРАВИЛЬНО
start.cmd
```

---

## Development Workflow

1. Изменения в коде
2. Запуск `start.cmd` для сборки
3. Проверка ошибок компиляции
4. Повторить

## CI/CD

В CI/CD пайплайнах также используйте `start.cmd` перед любыми cargo командами.
