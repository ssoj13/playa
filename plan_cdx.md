# plan_cdx

1. Починить загрузку кадров: File-comp возвращает placeholder вне диапазона и строит корректный путь к файлу; очередь пропускает фреймы без файла, логов об ошибке меньше.
2. Пересчёт таймлайна: `rebound()` считает min/max детей (допуская отрицательные стартовые кадры) и дергается при активации; если play_range совпадал с полным диапазоном, сбрасываем его на новый.
3. DnD/визуализация: использовать единый helper превью при переносе слоя (из проекта и внутри таймлайна), полоска-подсказка следует за мышью по вертикали.
4. Документация: обновить `docs/frame_flow.md` с цепочкой событий и найденными рисками; держать `AGENTS.md` актуальным для вклада.
5. Проверки: `./start.cmd` и точечный `cargo test` при изменении математики; собрать открытые вопросы по UX/производительности.
